# Documentation Chinese Translation Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Translate the repository's Markdown documentation into Chinese while preserving code, commands, links, and file structure.

**Architecture:** Treat the documentation corpus as a content-only migration. Establish repository-level guidance first, then translate the top-level entrypoint and chapter documents in batches while preserving Markdown anchors, code fences, and asset paths. Finish with consistency scans for leftover English text and link integrity.

**Tech Stack:** Git worktrees, Markdown, ripgrep, git status
